# Security Policy

## Reporting a Vulnerability

If you believe you have found a security vulnerability in this project, please report it to us as soon as possible. We take security seriously and will respond promptly to any reports.

1. **Email**: Please send an email to [security@oau-exam-backend.example.com](mailto:security@oau-exam-backend.example.com) with the details of the vulnerability.
2. **Details to Include**:
   - A description of the vulnerability.
   - Steps to reproduce the issue.
   - Any related links or code snippets if applicable.
3. **Confidentiality**: Please do not disclose the vulnerability publicly until it has been resolved.

## Security Best Practices

- Ensure your environment is configured securely.
- Regularly update dependencies to include security patches.
- Follow secure coding guidelines to minimize vulnerabilities.

Thank you for helping keep our project secure!
